# 300 历史与文明索引

> 变量体系切换：
>
> 从：
> 「叙事、年代、命运」
>
> 切换为：
> **「系统日志、相变、轨迹」**

---

## 1.【历史 History】

*   **传统定义**：对过去事件的研究。
*   **EMIS 定义**：文明操作系统的可读**执行日志 (Execution Log)**。
*   **EMIS 伪代码**：
    ```python
    History = System.Log(Time, Events, State_Changes)
    ```

## 2.【文明 Civilization】

*   **传统定义**：人类社会发展的先进阶段。
*   **EMIS 定义**：一种能够长时间维持高能量通量的**大型耗散结构 (Dissipative Structure)**。
*   **EMIS 伪代码**：
    ```typescript
    class Civilization extends Structure {
        energy_capture: high;
        duration: long;
        complexity: high;
    }
    ```

## 3.【时间 Time】(历史时间)

*   **传统定义**：存在的持续进程。
*   **EMIS 定义**：**熵积累**的不可逆轴线（历史之箭）。
*   **EMIS 伪代码**：
    ```python
    Time_Arrow = d(Total_Entropy) / dt > 0
    ```

## 4.【进步 Progress】

*   **传统定义**：向更好状态的发展。
*   **EMIS 定义**：**能量捕获密度**和结构复杂度的上升轨迹。
*   **EMIS 伪代码**：
    ```python
    Progress = d(Energy_Density) / dt > 0
    ```

## 5.【衰退 Decline】

*   **传统定义**：力量或数量的逐渐丧失。
*   **EMIS 定义**：当结构维护成本超过能量输入时（**EROI 为负**）。
*   **EMIS 伪代码**：
    ```python
    Decline = Maintenance_Cost > Energy_Input
    ```

## 6.【崩溃 Collapse】

*   **传统定义**：文明的突然失败。
*   **EMIS 定义**：因复杂度不可持续，结构被迫快速**简化**至低能态。
*   **EMIS 伪代码**：
    ```python
    Collapse = Complexity.reduce(Target_Level=Low)
    # 为了保存内核的强制重启
    ```

## 7.【帝国 Empire】

*   **传统定义**：统一权威下的国家群。
*   **EMIS 定义**：通过强制扩张最大化**能量榨取半径**的结构。
*   **EMIS 伪代码**：
    ```python
    Empire = Maximize(Territory_Radius) where Coercion_Cost < Extraction_Yield
    ```

## 8.【时代 Era / Epoch】

*   **传统定义**：一段独特的历史时期。
*   **EMIS 定义**：由稳定的**主导能量源**（如煤炭时代）定义的离散时间块。
*   **EMIS 伪代码**：
    ```python
    Era = Time_Range(Energy_Source == Constant)
    ```

## 9.【事件 Event】

*   **传统定义**：发生的事情。
*   **EMIS 定义**：记录在系统日志中的特定**状态改变 (State Change)**。
*   **EMIS 伪代码**：
    ```python
    Event = System_State(t) != System_State(t-1)
    ```

## 10.【因果性 Causality】

*   **传统定义**：因果关系。
*   **EMIS 定义**：导致状态改变的能量/信息约束的**逻辑链**。
*   **EMIS 伪代码**：
    ```python
    Causality = Constraint_A.trigger(Constraint_B)
    ```

## 11.【偶然性 Contingency】

*   **传统定义**：可能发生但不确定的事。
*   **EMIS 定义**：决定性约束场内的**随机噪声 (Noise)** 或分叉点。
*   **EMIS 伪代码**：
    ```python
    Contingency = Noise(System_Trajectory)
    ```

## 12.【趋势 Trend】

*   **传统定义**：变化的大致方向。
*   **EMIS 定义**：个体寻能行为的**矢量和 (Vector Sum)**。
*   **EMIS 伪代码**：
    ```python
    Trend = Sum(All_Agent_Vectors)
    ```

## 13.【传统 Tradition】

*   **传统定义**：代代相传的习俗。
*   **EMIS 定义**：曾经节能有效的**遗留代码 (Legacy Code)** 或缓存配置。
*   **EMIS 伪代码**：
    ```python
    Tradition = Cache.read(Old_Config)
    # 警告：可能已弃用
    ```

## 14.【复兴 Renaissance】

*   **传统定义**：艺术和文学的复兴。
*   **EMIS 定义**：从旧的高效结构备份中进行**系统还原 (System Restore)**。
*   **EMIS 伪代码**：
    ```python
    Renaissance = System.load_backup("Classical_Era")
    ```

## 15.【黑暗时代 Dark Age】

*   **传统定义**：文化和经济衰退期。
*   **EMIS 定义**：**信息连接度极低**且能量网络碎片化的时期。
*   **EMIS 伪代码**：
    ```python
    Dark_Age = Network.fragment() AND Information_Flow ≈ 0
    ```

## 16.【路径依赖 Path Dependence】

*   **传统定义**：历史很重要。
*   **EMIS 定义**：当前结构选项受制于**过往能量投资**（沉没成本）。
*   **EMIS 伪代码**：
    ```python
    Current_State.constraints += Previous_State.structure
    ```

## 17.【奇点 Singularity】

*   **传统定义**：快速增长的时间点。
*   **EMIS 定义**：**能量-信息转化速度**趋近于无穷大的点。
*   **EMIS 伪代码**：
    ```python
    Singularity = lim(d(Info)/dt) -> ∞
    ```

## 18.【档案 Archive】

*   **传统定义**：历史文件的集合。
*   **EMIS 定义**：文明系统的**外部存储器 (External Storage)**。
*   **EMIS 伪代码**：
    ```python
    Archive = External_Storage.write(System_Log)
    ```

## 19.【史学 Historiography】

*   **传统定义**：对历史写作的研究。
*   **EMIS 定义**：对生成系统日志所用的**压缩算法**的分析。
*   **EMIS 伪代码**：
    ```python
    Historiography = Analyze(Log_Compression_Method)
    ```

## 20.【演化 Evolution】(社会)

*   **传统定义**：社会的逐渐发展。
*   **EMIS 定义**：寻找稳定能量配置的迭代**优化算法 (Optimization Algorithm)**。
*   **EMIS 伪代码**：
    ```python
    while True:
        Society.optimize(Energy_Efficiency)
    ```
