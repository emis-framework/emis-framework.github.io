# 500 地理学索引

> 变量体系切换：
>
> 从：
> 「地点、区域、边界」
>
> 切换为：
> **「坐标、能量密度、摩擦区」**

---

## 1.【地理 Geography】

*   **传统定义**：研究人与环境关系的学科。
*   **EMIS 定义**：对行星表面的**物质分布**和**时空约束**的研究。
*   **EMIS 伪代码**：
    ```python
    Geography = Map(Matter_Distribution, Energy_Sources)
    ```

## 2.【空间 Space】

*   **传统定义**：连续的区域。
*   **EMIS 定义**：所有物质和能量流的**容器 (Container)**；穿越它有成本。
*   **EMIS 伪代码**：
    ```python
    Space = Matrix(Distance_Cost)
    ```

## 3.【领土 Territory】

*   **传统定义**：管辖区域。
*   **EMIS 定义**：具有**独家访问控制 (Exclusive Access Control)** 的定义空间区域。
*   **EMIS 伪代码**：
    ```python
    Territory = Zone.lock(Owner)
    ```

## 4.【边界 Border】

*   **传统定义**：分隔政治区域的线。
*   **EMIS 定义**：能量/信息流动阻力最大化的**高阻抗线 (High-Impedance Line)**。
*   **EMIS 伪代码**：
    ```python
    Border = Line(Resistance=Max)
    ```

## 5.【资源 Resource】

*   **传统定义**：材料或资产储备。
*   **EMIS 定义**：物质中局部的**能量集中 (负熵)**。
*   **EMIS 伪代码**：
    ```python
    Resource = Local_Stock(Energy > Average)
    ```

## 6.【环境 Environment】

*   **传统定义**：周围环境。
*   **EMIS 定义**：热力学系统的**边界条件 (Boundary Conditions)**（温度、压力）。
*   **EMIS 伪代码**：
    ```python
    Environment = System.Boundary_Conditions
    ```

## 7.【气候 Climate】

*   **传统定义**：区域的普遍天气。
*   **EMIS 定义**：**太阳能分布曲线**及其波动方差。
*   **EMIS 伪代码**：
    ```python
    Climate = Distribution(Solar_Flux, Water_Cycle)
    ```

## 8.【距离 Distance】

*   **传统定义**：物体间的空间量。
*   **EMIS 定义**：将物质/信息从 A 移动到 B 所需的**能量成本**。
*   **EMIS 伪代码**：
    ```python
    Distance = Work / Force
    # 在 EMIS 中，距离单位是焦耳
    ```

## 9.【物流 Logistics】

*   **传统定义**：复杂操作的组织。
*   **EMIS 定义**：跨越物理空间的**能量流路径优化**。
*   **EMIS 伪代码**：
    ```python
    Logistics = Minimize(Transport_Energy_Cost)
    ```

## 10.【城市化 Urbanization】

*   **传统定义**：人口向城市转移。
*   **EMIS 定义**：节点为了最小化连接成本而进行的空间**致密化 (Densification)** (缩放定律)。
*   **EMIS 伪代码**：
    ```python
    Urbanization = Densify(Nodes) -> Superlinear_Output
    ```

## 11.【核心-边缘 Core-Periphery】

*   **传统定义**：发达地区与发展中地区的关系。
*   **EMIS 定义**：**榨取拓扑结构**。能量从低结构的边缘流向高结构的核心。
*   **EMIS 伪代码**：
    ```python
    Flow_Vector = Periphery -> Core
    ```

## 12.【地缘政治 Geopolitics】

*   **传统定义**：受地理影响的政治。
*   **EMIS 定义**：在资源分布不均的地图上进行的**博弈论 (Game Theory)**。
*   **EMIS 伪代码**：
    ```python
    Geopolitics = Game(Map_Constraints, Resource_Locations)
    ```

## 13.【咽喉点 Chokepoint】

*   **传统定义**：拥堵或阻塞点。
*   **EMIS 定义**：具有**关键拓扑**的空间节点，在此处控制流量只需极小能量。
*   **EMIS 伪代码**：
    ```python
    Chokepoint = Node(Control_Leverage=Max)
    # 例如：苏伊士运河
    ```

## 14.【移民 Migration】

*   **传统定义**：人口移动。
*   **EMIS 定义**：由能量梯度驱动的**粒子扩散 (Particle Diffusion)**。
*   **EMIS 伪代码**：
    ```python
    Migration_Vector = Gradient(Energy_Opportunity)
    ```

## 15.【景观 Landscape】

*   **传统定义**：土地的可见特征。
*   **EMIS 定义**：影响移动成本的**摩擦表面 (Friction Surface)**。
*   **EMIS 伪代码**：
    ```python
    Landscape = Surface(Friction_Coefficient)
    ```

## 16.【生态系统 Ecosystem】

*   **传统定义**：生物群落。
*   **EMIS 定义**：生物智能体之间**能量交换循环**的平衡网络。
*   **EMIS 伪代码**：
    ```python
    Ecosystem = Network(Trophic_Levels, Recycling_Loops)
    ```

## 17.【承载力 Carrying Capacity】

*   **传统定义**：环境能维持的最大人口。
*   **EMIS 定义**：可用于支持生物量结构的能量通量**上限 (Upper Limit)**。
*   **EMIS 伪代码**：
    ```python
    Limit = Total_Solar_Input * Conversion_Efficiency
    ```

## 18.【供应链 Supply Chain】

*   **传统定义**：生产商品的流程序列。
*   **EMIS 定义**：跨越地理空间的序列化**增值结构**。
*   **EMIS 伪代码**：
    ```python
    Chain = Link(Extraction -> Processing -> Distribution)
    ```

## 19.【腹地 Hinterland】

*   **传统定义**：海岸或河流后的偏远地区。
*   **EMIS 定义**：向中心节点（城市/港口）供能的**集能区 (Catchment Area)**。
*   **EMIS 伪代码**：
    ```python
    Hinterland = Area(Output -> Center_Node)
    ```

## 20.【人类世 Anthropocene】

*   **传统定义**：人类影响地质的时代。
*   **EMIS 定义**：**人类能量耗散**超过自然地质背景噪声的时代。
*   **EMIS 伪代码**：
    ```python
    Anthropocene = Human_Energy_Flux > Geo_Energy_Flux
    ```
